# Paper Figures

This directory contains curated figures for the paper.

- `drift_eps_sweep_combined.png`: residual-stream drift plot for the epsilon
  sweep.
- `llama_pareto_post_arditi.pdf` / `.png`: Llama Pareto figure from the earlier
  Arditi-focused sweep.

Future NeurIPS-level sweep figures should be exported here, while full raw
Optuna CSVs and logs should remain under `results/`.
